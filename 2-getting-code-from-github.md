# Getting code from GitHub

**Method 1:** Cloning

* Visit the repo you want. ex: https://github.com/jpweeks/particulate-js
* Copy the ssh URL
* In terminal, do:

````
git clone <the-url-you-copied>
```` 

**Method 2:** Download zip file

* Visit the repo you want. ex: https://github.com/jpweeks/particulate-js
* Hit the [Download ZIP] button
